# Проверка решений квадратных уравненийПрограмма для моделирования процесса проверки студенческих работ по решению квадратных уравнений.## ОписаниеПрограмма имитирует реальный процесс:1. Читает квадратные уравнения из файла `equations.txt`2. Раздает задачи студентам разных уровней подготовки3. Студенты решают уравнения 4. Преподаватель проверяет ответы5. Выводит итоговую таблицу успеваемости## Типы студентов**GOOD - Всегда решает правильно **MID - 50% вероятность правильного решения **BAD - Всегда отвечает, что корень один и это 0 ## Формат входного файлаФайл `equations.txt` содержит коэффициенты квадратных уравнений в формате:Результат: Загружено уравнений: 15Сгенерировано писем: 75Проверено писем: 75=== Итоги зачетной недели ===Сидоров             | 2Петров              | 6Смирнов             | 15Козлов              | 9Иванов              | 15
